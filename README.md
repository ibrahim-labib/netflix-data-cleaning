# Netflix Dataset Cleaning Project

## Overview
Cleaned and structured a real-world Netflix dataset using Python (Pandas). Handled missing values, standardized text data, and converted date formats.

## Steps
- Data loading and exploration
- Missing value handling
- Data cleaning and formatting
- Date conversion
- Final dataset export

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Output
Clean dataset ready for analysis and visualization.
